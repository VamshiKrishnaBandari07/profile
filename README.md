# Vamshi Krishna Bandari — Portfolio

Professional 3D portfolio website built with **Three.js** and **Vite**.

**Live site:** https://vamshikrishnabandari07.github.io/profile/

---

## Features

- Interactive **3D particle background** with wireframe geometry
- **Featured Highlights** gallery — Parliament AI Showcase, London Tech, FRC Hackathon, DonorLink
- Responsive dark-theme design with glassmorphism
- Scroll-reveal animations
- GitHub Pages auto-deploy via Actions

---

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:5173/profile/

---

## Replace Highlight Photos with Your LinkedIn Images

1. Open your [LinkedIn activity](https://www.linkedin.com/in/vamshi-krishna-bandari-623580212/recent-activity/all/)
2. Find posts for:
   - **Parliament** — AI in Healthcare Parliamentary Showcase
   - **London Tech** — your tech summit / innovation event photos
   - **FRC Hackathon** — FRC & ODI Innovation Sprint
   - **DonorLink** — project screenshot
3. Save images and replace files in `public/assets/highlights/`:

| File | Event |
|------|-------|
| `parliament.jpg` | UK Parliament AI Healthcare Showcase |
| `london-tech.jpg` | London Tech Summit / innovation event |
| `frc-hackathon.jpg` | FRC & ODI Innovation Sprint |
| `donorlink.jpg` | DonorLink platform screenshot |

4. Commit and push — GitHub Actions will redeploy automatically.

---

## GitHub Pages Setup

1. Repo **Settings → Pages**
2. Source: **GitHub Actions**
3. Push to `main` — workflow deploys automatically
